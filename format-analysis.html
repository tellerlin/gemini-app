📊核心特殊格式支持

🧜‍♀️ Mermaid 流程图 ✅ 完全支持

功能：支持各种流程图、时序图、类图等，特别优化了中文字符渲染

语法：
```mermaid
graph TD
    A["开始"] --> B{"判断条件?"}
    B --> C["处理结果"]
```
技术栈：
  • 🔧 OptimizedMermaidDiagram 组件
  • 🔧 fixMermaidSyntax 函数（中文优化）
  • 🔧 mermaid.js 库
  • 🔧 导出功能：SVG、PDF

特殊优化：

  • ✅ 中文字符自动双引号包装
  • ✅ 分号和注释自动清理
  • ✅ 中文标点符号转换
  • ✅ 缩放和导出功能

🧮 数学公式 ✅ 完全支持

功能：支持内联和块级数学公式，使用 KaTeX 渲染

语法：
内联公式：$E = mc^2$
块级公式：$$\\int_{-\\infty}^{\\infty} e^{-x^2} dx = \\sqrt{\\pi}$$
技术栈：
  • 🔧 remarkMath 插件
  • 🔧 rehypeKatex 插件
  • 🔧 KaTeX CSS 自动导入

💻 代码高亮 ✅ 完全支持

功能：支持100+编程语言的语法高亮

语法：
```javascript
function hello() {
    console.log("Hello World!");
}
```
技术栈：
  • 🔧 react-syntax-highlighter
  • 🔧 CodeBlockCopy 组件（复制功能）
  • 🔧 Prism.js 主题

📈高级数据可视化

📊 图表组件 🚀 高级功能

功能：支持多种交互式图表类型

语法：
```chart
{
  "type": "line",
  "data": [{"x": 1, "y": 10}, {"x": 2, "y": 20}],
  "config": {"title": "示例图表"}
}
```
支持的图表类型：
📈 line - 折线图
📊 bar - 柱状图
🥧 pie - 饼图
📄 area - 面积图
🎯 radar - 雷达图
🔗 composed - 组合图
⚪ scatter - 散点图
技术栈：
  • 🔧 ChartComponent 主组件
  • 🔧 recharts 图表库
  • 🔧 导出功能：SVG、PDF
  • 🔧 动态类型切换

📋 增强表格 🚀 高级功能

🎯使用建议

💡 最佳实践

  1. Mermaid图表：使用中文时系统会自动优化，无需手动添加引号
  2. 数学公式：内联用单 $，块级用双 $$
  3. 表格数据：使用JSON格式，支持数值排序
  4. 图表配置：可以指定颜色、样式等详细配置
  5. 代码高亮：指定正确的语言标识符获得最佳效果
